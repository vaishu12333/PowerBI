IPL Analysis Dashboard (2008–2025)

About Project

I created this Power BI dashboard to analyze IPL data from 2008 to 2025. The dashboard shows season-wise team performance, winners, player stats, and match records in a simple visual format.

Main Features

- Season wise filter
- Champion and Runner-up team
- Total matches played
- Total 4s and 6s
- Orange Cap holder
- Purple Cap holder
- Points table
- Total venues
- Centuries and Half-centuries

Tools Used

- Power BI
- Excel / CSV file
- Data cleaning
- DAX formulas

Dashboard Benefits

This dashboard helps to understand IPL trends, best performing teams, top players, and season comparisons quickly.

Files in Project

- IPL Dashboard.pbix
- IPL Dashboard.png
- Dataset file
- README.md

How to Open

1. Download project files
2. Open ".pbix" file in Power BI Desktop
3. Select season from filter
4. View dashboard insights

Purpose

This project is useful for learning Power BI, data visualization, and sports analytics.
